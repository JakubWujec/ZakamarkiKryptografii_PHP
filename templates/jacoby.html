Symbol Jacobiego

Definicja

Niech $p$ będzie nieparzystą liczbą pierwszą a $a$ liczbą całkowitą. Symbol Legendre'a $ \left(\frac{a}{p}\right)$ jest zdefiniowany jako: $$ \left(\frac{a}{p}\right) = \left\{ \begin{array}{ll} 0 & \textrm{ jeżeli } p|a \\ 1 & \textrm{ jeżeli } a \in \mathbb{Q}_{p}\\ -1 & \textrm{ jeżeli } a \in \overline{\mathbb{Q}_{p}} \end{array} \right. $$

Własności symbolu Legendre'a

Niech $a,b \in \mathbb{Z}$, zaś $p$ to nieparzysta liczba pierwsza. Wówczas:
  • $ \left(\frac{a}{p}\right) \equiv_p a^{\frac{(p-1)}{2}} $
  • $ \left(\frac{ab}{p}\right) = \left(\frac{a}{p}\right)\left(\frac{b}{p}\right) $
  • $ a \equiv_p b \Longrightarrow \left(\frac{a}{p}\right) = \left(\frac{b}{p}\right)$
  • $ \left(\frac{2}{p}\right) = (-1)^{\frac{(p^2-1)}{8}} $
  • Jeżeli $q$ jest nieparzystą liczba pierwszą inna od p to: $$ \left(\frac{p}{q}\right) = \left(\frac{q}{p}\right)(-1)^{\frac{(p-1)(q-1)}{4}} $$

Definicja

Niech $ n \geq 3 $ bedzie liczbą nieparzystą a jej rozkład na czynniki pierwsze to $ n = p_1^{e_1}p_2^{e_2}\dots p_k^{e_k}$ Symbol Jacobiego $(\frac{a}{n})$ jest zdefiniowany jako: $$ \left(\frac{a}{n}\right) = \left(\frac{a}{p_1}\right)^{e_1} \left(\frac{a}{p_2}\right)^{e_2} \dots \left(\frac{a}{p_k}\right)^{e_k} $$ Jeżeli $n$ jest liczbą pierwszą, to symbol Jacobiego jest symbolem Legendre'a.

Własności symbolu Jacobiego.

Niech $a,b \in \mathbb{Z}$ zaś $m,n \geq 3$ to nieparzyste liczby całkowie. Wówczas:

  • $ \left(\frac{a}{n}\right) = 0,1 \textrm{ albo } -1 \textrm{ Ponadto} \left(\frac{a}{n}\right) = 0 \Longleftrightarrow \gcd{(a,n)} \neq 1 $
  • $ \left(\frac{ab}{n}\right) = \left(\frac{a}{n}\right) \left(\frac{b}{n}\right) $
  • $ \left(\frac{a}{mn}\right) = \left(\frac{a}{m}\right) \left(\frac{a}{n}\right) $
  • $ a \equiv_n b \longrightarrow \left(\frac{a}{n}\right) = \left(\frac{b}{n}\right) $
  • $ \left(\frac{1}{n}\right) = 1 $
  • $ \left(\frac{-1}{n}\right) = (-1)^{\frac{(n-1)}{2}} $
  • $ \left(\frac{2}{n}\right) = (-1)^{\frac{(n^2-1)}{8}} $
  • $ \left(\frac{m}{n}\right) = \left(\frac{n}{m}\right)(-1)^{\frac{(m-1)(n-1)}{4}} $

Z własności symbolu Jacobiego wynika, że jeżeli $n$ nieparzyste oraz $a$ nieparzyste i w postaci $ a = 2^ea_1$, gdzie $a_1$ też nieparzyste to: $$ \left(\frac{a}{n}\right) = \left(\frac{2^e}{n}\right) \left(\frac{a_1}{n}\right) = \left(\frac{2}{n}\right)^e \left(\frac{n \mod a_1}{a_1}\right) (-1)^{\frac{(a_1-1)(n-1)}{4}} $$

Algorytm obliczania symbolu Jacobiego $\left(\frac{a}{n}\right)$ (i Legendre'a) dla nieparzystej liczby całkowitej $ n \geq 3 $ oraz całkowitego $ 0 \leq a \le n $

$\textrm{JACOBI} (a,n)$ If $a = 0$ then return 0 If $a = 1$ then return 1 Write $ a = 2^ea_1 $, gdzie $a_1$ nieparzyste If $e$ parzyste $ s \leftarrow 1 $ else if $ n \equiv_8 1 \textrm{ or } 7 $ $ s \leftarrow 1 $ else if $ n \equiv_8 3 \textrm{ or } 5 $ $ s \leftarrow -1 $ If $n \equiv_4 3 \textrm{ and } a_1 \equiv_4 3 $ $ s \leftarrow -s $ $ n_1 \leftarrow n\mod a_1 $ If $ a_1 = 1 $ then return $s$ else return $s*JACOBI(n_1,a_1)$ Algorytm działa w czasie $\mathcal{O}((\log{n})^2)$
